# GraphPlotter
A simplified platform for students to plot a graph based on various algebric , logarithmic and exponential functions.

Simple desktop app that lets students type a single function expression (like a search box) and plots it. Supports polynomials, trig and common numpy functions. Built with Tkinter + matplotlib + NumPy.

Features
- Single-line input bar for expression (very easy for students).
- Domain controls: x start, x end, number of points.
- Quick example expressions.
- Plots function curve and handles scalar or array results.
- Clear button to close plots.

Requirements
- Python 3.8+
- numpy
- matplotlib
- tkinter (usually included with standard Python on Windows)
